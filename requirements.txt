# ========================================
# SYSTEM REQUIREMENTS & NOTES
# ========================================

# PLATFORM-SPECIFIC REQUIREMENTS:
# 
# Linux/Kali Linux:
#   - Requires root privileges (sudo) for raw socket operations
#   - libpcap-dev package may be required
#   - Installation: sudo apt-get install libpcap-dev
#
# Windows Systems:
#   - WinPcap or Npcap driver installation required
#   - Download from: https://nmap.org/npcap/
#   - May require Administrator privileges
#
# macOS/Darwin:
#   - Requires elevated privileges (sudo) for packet capture
#   - Homebrew installation recommended for dependencies
#   - brew install libpcap
#
# Python Version:
#   - Minimum: Python 3.6
#   - Recommended: Python 3.8+
#   - Not compatible with Python 2.x

# ========================================
# INSTALLATION INSTRUCTIONS
# ========================================

# Standard installation:
#   pip install -r requirements.txt
#
# With elevated privileges (Linux/macOS):
#   sudo pip3 install -r requirements.txt
#
# Development installation (includes testing tools):
#   pip install -r requirements.txt -r requirements-dev.txt
#
# Specific package installation:
#   pip install scapy==2.4.5
#   pip install psutil==5.9.0
#
# Upgrade all packages:
#   pip install --upgrade -r requirements.txt

# ========================================
# TROUBLESHOOTING
# ========================================

# Permission Issues:
#   - Run with: sudo python3 network_traffic_analyzer.py
#   - Or grant capabilities: setcap cap_net_raw=ep /usr/bin/python3
#
# Missing libpcap:
#   Linux: sudo apt-get install libpcap-dev
#   macOS: brew install libpcap
#
# Scapy not found:
#   - Verify installation: python3 -c "import scapy; print(scapy.__version__)"
#   - Reinstall: pip install --force-reinstall scapy
#
# Network interface not detected:
#   - List available: ip link show (Linux)
#   - Check with: sudo python3 network_traffic_analyzer.py --list-interfaces
